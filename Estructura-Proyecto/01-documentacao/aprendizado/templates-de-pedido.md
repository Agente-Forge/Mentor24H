# Templates de Pedido — Mentor24h
**Projeto:** Mentor24h | **Modo Aprendiz:** PAE v2.0
*Como pedir cada tipo de coisa para o PAE de forma precisa*

> Use estes templates como ponto de partida.
> Preencha os [ ] com o que você quer. O PAE entende o resto.

---

## 1. Nova Feature / Funcionalidade

Use quando: quiser **criar algo novo** que ainda não existe no app.

```
"Quero criar [o quê] no módulo [nome do módulo]
 com visual [Apple / Apple+]
 [detalhe opcional: o que o usuário faz, o que aparece na tela]"
```

**Exemplo:**
> "Quero criar um sistema de favoritos no módulo Contatos com visual Apple+
>  — o usuário clica numa estrela e o contato vai para uma lista de favoritos"

---

## 2. Substituir / Remover algo existente

Use quando: quiser **trocar** uma coisa por outra, ou **remover** algo que não quer mais.

```
"Quero substituir [o quê] por [o quê novo] no módulo [nome]
 [detalhe opcional: comportamento esperado do novo]"
```

**Exemplo:**
> "Quero substituir a view Kanban por um sistema de Temperatura no módulo Contatos
>  — 6 status coloridos: VIP, Quente, Morno, Lead, Frio, Inativo"

---

## 3. Bugfix — Corrigir algo que não está funcionando

Use quando: algo **está quebrando**, aparecendo errado, ou não funcionando como deveria.

```
"Tem um bug no módulo [nome]:
 [descreva o que acontece] em [desktop / mobile / ambos]
 [opcional: quando exatamente acontece]"
```

**Exemplo:**
> "Tem um bug no módulo Contatos: o filtro de busca quebra o layout no mobile
>  — quando digito mais de 3 letras a tela fica estranha"

---

## 4. Redesign Visual

Use quando: o funcionamento está **ok** mas quer deixar **mais bonito, moderno, ou diferente visualmente**.

```
"Quero melhorar o visual de [o quê] no módulo [nome]
 [detalhe: o que não gosta / o que quer ver]"
```

**Exemplo:**
> "Quero melhorar o visual dos cards de contato no módulo Contatos
>  — quero que fiquem mais espaçados, com foto maior e com a temperatura bem visível"

---

## Dica: Como descrever o que você vê na tela

Se você não sabe o nome técnico, descreva assim:

| O que você vê | Nome técnico |
|---------------|--------------|
| "aquela janela que abre por cima" | modal |
| "aquela barrinha de cima com o nome do app" | header / navbar |
| "aquelas abinhas para trocar de seção" | tabs |
| "aquela etiquetinha colorida" | badge |
| "aquela grade de quadradinhos" | grid / cards |
| "aquela listinha com linhas" | lista / list view |
| "aquela mensagem que aparece e some" | toast / notificação |
| "aquele botãozinho de ação rápida" | action button / FAB |
| "aquela coluna que desliza pelo lado" | sidebar / drawer |
