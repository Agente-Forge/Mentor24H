## 🎯 Fluxo FORGE (12 Skills Orquestradas)

Este projeto segue a **metodologia FORGE** — uma "Digital Production House" onde cada skill tem um papel:

```
[1] skill-orquestrador    → Entrevista inicial + coordenação
[2] skill-consultor       → PRD + CONSTITUTION
[3] skill-planner         → Sprints + Roadmap
[4] skill-scaffolder      → Estrutura (você está aqui ✅)
[5a,5b,5c] PARALELO       → Docs + Design + Security
[6] skill-construtor      → Desenvolvimento TDD
[6s] skill-sentinela      → Code review real-time
[7] skill-performance     → Auditoria performance
[8] skill-devops          → Deploy + CI/CD
[9] skill-health-monitor  → Manutenção semanal
```
