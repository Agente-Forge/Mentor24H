🔐 Segurança & Compliance
⚠️ Risco Crítico: API keys LLM em localStorage (plaintext). Status: Mitigado com CSP + aviso visual + função friendlyApiError()
✅ Implementado: CSP meta-tag (SEC-01) bloqueia exfiltração localStorage

🛡️ Medidas Implementadas

  • SEC-01: Content Security Policy (default-src 'self', connect-src whitelisted)
  • SEC-02: XSS Protection — todas as renderizações com esc()
  • SEC-03: Modal ao invés de prompt() — consistência UX
  • SEC-05: Aviso visual ao salvar API key — "⚠️ Armazenado em localStorage"
  • SEC-06/07: Google Fonts CSP allowlist
  • SEC-08/09/10/11: Provider whitelisting (OpenAI, Groq, etc.)
